#                       5G MAC Layer Simulator 
This project simulates a simplified MAC(Medium Access Control) layer of a 5G communication system.
# Features
> Packet segmentation
> Queue-based transmission
> Random packet loss simulation 
> Retransmission of lost packets
> Network delay simulation 
> Summary statistics

# Technologies used
> C Programming
> Data Structure (Queue)
> GCC Complier

# How to Run

```bash
gcc hello.c queue.c -o simulator
.\simulator
```
